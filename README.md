# Onboarding Metrics Dashboard

Live at: https://urbanpiperimplementation.netlify.app/

## Files
- `index.html` - Main entry point with React bundle loader
- `bundle.js` - Compiled React app (1.2 MB)
- `styles.css` - Application styles
- `ttgl_data.json` - TTGL metrics data
- `revrec_data.json` - Revenue realisation cohort data
- `om_performance.json` - OM performance analytics
- `netlify.toml` - Netlify configuration

## Data Refresh
Data files are refreshed daily at 8:00 AM IST via GitHub Actions.
